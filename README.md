# 平面圖管理系統

一個用於管理和查看平面圖的工具。

## 功能特點

- 支持 PDF 和圖片格式的平面圖
- 可以在平面圖上添加標記點
- 支持為每個點添加描述、到期日期和附件
- 自動提醒即將到期的項目

## 安裝說明

1. 下載最新版本的安裝包
2. 解壓縮下載的文件
3. 運行 `安裝.bat`
4. 程序將安裝到用戶文檔目錄
5. 安裝完成後，桌面會出現快捷方式

## 系統要求

- Windows 10 或更高版本
- 不需要管理員權限
